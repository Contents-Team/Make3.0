# Plans

> 진행 중 계획서. 확정 결정은 `../adr/` 로, 완료 후엔 `CHANGELOG.md` 로 승격.

## Conventions

- 파일명: `YYYY-MM-DD-short-slug.md`
- 첫 줄에 `Status` (`Draft` / `Active` / `Done` / `Abandoned`) + 1줄 요약
- 작업 완료 시 ADR 또는 `CHANGELOG.md` 로 정리 후 삭제 가능

## Section Headers (Convention)

각 계획서는 다음 영문 헤더를 사용 (필요한 것만):

- `## Context`
- `## Goals`
- `## Non-goals`
- `## Refactor plan`
- `## Constraints`
- `## Risks`
- `## Rollout`
